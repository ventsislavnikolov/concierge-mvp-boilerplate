      Connect to GitHub

      The view reads the map's issues directly from the GitHub API. For a public repo a token is optional — leave it blank to browse read-only. Any token you add stays in this browser (localStorage) and is sent only to api.github.com.
      Public repo → no token needed (60 API calls/hr shared). Each load is ~40 calls, so if you refresh often add a fine-grained PAT from github.com/settings/personal-access-tokens (Issues: Read-only) to get 5,000/hr.
